## Обзор
Crypto API Automation Toolkit — модульный SDK и набор утилит для автоматизации взаимодействия с официальными API криптовалютных бирж (market data, order management, account info). Идеален для разработчиков и трейдеров, которым нужны быстрые и удобные примеры, надёжная обёртка API и инструменты для проверки лимитов запросов.

Основные сценарии:
- Автоматизированные торговые роботы (исполнение ордеров через официальные API)
- Сбор и агрегация рыночных данных
- Тестирование ограничений по API (rate limits)
- Многоучётная автоматизация и прокси-поддержка

---

## Возможности
- Поддержка Python (3.8+) и Node.js (14+)
- Готовые примеры для быстрых стартов
- Proxy: HTTP(S) и SOCKS5
- Rate-limit tester (CLI) — измеряет throughput, latency, success rate
- Лёгкая модульная архитектура — добавить адаптер под другую биржу просто
- Конфигурация через `.env` / конфиг файлы
- Dockerfile для запуска в контейнере
- Документация и демонстрации в README + папка `assets/`

---

## Поддерживаемые языки и окружение
- Python: рекомендуемая версия 3.8+
- Node.js: рекомендуемая версия 14+ (LTS)
- Опционально: Docker (если хочешь запускать в контейнере)
- Работает на Windows / macOS / Linux
